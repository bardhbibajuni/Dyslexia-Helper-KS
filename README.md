# Dyslexia-Helper-KS
Dyslexia Helper makes websites easier to read by applying dyslexia-friendly settings across pages you visit.  Features: 1.Choose readable fonts (Lexend, OpenDyslexic, Comic Sans, Arial, or keep the original) 2.Adjust letter spacing and word spacing 3.Optional soft background colors to reduce eye strain 4.Quick presets (Default / Balanced / Strong)
